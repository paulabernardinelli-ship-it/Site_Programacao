# A Gata Selene - Curadoria Literária Inteligente

![Version](https://img.shields.io/badge/version-2.6-purple.svg)
![Status](https://img.shields.io/badge/status-active-success.svg)

> **Descubra seu próximo livro favorito com ajuda da inteligência artificial e curadoria humana**

Uma plataforma web elegante que combina tecnologia e paixão por literatura para conectar leitores aos livros perfeitos para seu gosto pessoal.

## ✨ Características Principais

###  Curadoria Inteligente
- **Busca Automática** no Google Books API
- **Foco em Lançamentos** (2020-2025)
- **Filtros por Gênero** e Autores Favoritos
- **Recomendações Personalizadas** baseadas no perfil

###  Design Premium
- **Interface Elegante** com tema roxo-dourado
- **Tipografia Refinada** (Cormorant + Nunito)
- **Layout Responsivo** para todos os dispositivos
- **Animações Suaves** e experiência imersiva

###  Funcionalidades Técnicas
- **Menu Fixo** consistente em todas as páginas
- **Busca de CEP** automática via ViaCEP
- **Validação de Formulários** em tempo real
- **Otimização de Performance** e SEO

##  Como Usar

### 1.  Cadastro
- Preencha seus dados pessoais
- Use a busca automática de endereço por CEP
- Informe suas preferências literárias iniciais

### 2.  Preferências
- Selecione seus gêneros literários favoritos
- Adicione autores preferidos
- Defina hábitos de leitura
- Descreva o que te cativa em uma história

### 3.  Descobrir Livros
- Receba recomendações personalizadas
- Explore lançamentos recentes e best-sellers
- Acesse informações detalhadas dos livros
- Veja diretamente no Google Books

##  Tecnologias Utilizadas

### Frontend
- **HTML5** - Estrutura semântica
- **CSS3** - Design moderno com Grid e Flexbox
- **JavaScript ES6+** - Interatividade e APIs

### APIs Integradas
- **Google Books API** - Busca de livros
- **ViaCEP** - Consulta de endereços
- **Google Fonts** - Tipografia premium

### Design System
- **Cores**: Roxo (#5a1ea6), Dourado (#f4c542), Rosa (#f7c7db)
- **Fontes**: Cormorant Garamond (títulos), Nunito (texto)
- **Layout**: Container de 1200px, totalmente responsivo
